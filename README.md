#  Secure Password Generator

A secure password generator built using Django that helps users generate strong, secure, and customizable passwords to enhance online security.

---

##  Features

* Generate secure and random passwords
* Customize password length
* Include uppercase letters (A-Z)
* Include lowercase letters (a-z)
* Include numbers (0-9)
* Include special characters (!@#$%^&*)
* User-friendly interface
* Fast password generation
* Responsive design

## ⚙️ Installation Guide

### 1. Clone the Repository

```bash
git clone https://github.com/OM0126/secure-password-generator.git
cd secure-password-generator
```

### 2. Create a Virtual Environment

```bash
python3 -m venv .venv
```

### 3. Activate the Virtual Environment

#### Linux / macOS

```bash
source .venv/bin/activate
```

#### Windows

```bash
.venv\Scripts\activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Run Database Migrations

```bash
python manage.py migrate
```

### 6. Start the Development Server

```bash
python manage.py runserver
```

### 7. Open the Application

Visit:

```text
http://127.0.0.1:8000/
```

## Why Use Strong Passwords?
Strong passwords help protect accounts from:

* Brute-force attacks
* Dictionary attacks
* Credential stuffing
* Unauthorized access

A secure password should:

* Be at least 12 characters long
* Contain uppercase and lowercase letters
* Include numbers
* Include special characters
* Be unique for every account
